## Was enthält das Spiel?

| Feature | Details |
|---|---|
| **Vokabeln** | 30 Begriffe aus 4 Kategorien |
| **Level** | 3 Schwierigkeitsstufen |
| **Modi** | Karteikarten · Spanisch→Deutsch · Deutsch→Spanisch · Eintippen |
| **Feedback** | Direkt nach jeder Frage |
| **Belohnung** | Konfetti + Trophäe bei ≥ 70% |
| **Gamification** | Leben · Streak · XP |

## Vokabular-Kategorien

- **Politik** — dictadura, golpe de estado, exilio, represión, desaparecido, resistencia …
- **Gesellschaft** — solidaridad, memoria colectiva, identidad nacional, testimonio …
- **Literatur** — narrador, metáfora, compromiso político, realismo mágico, símbolo …
- **Chile** — Unidad Popular, DINA, Estadio Nacional, Nueva Canción Chilena …
